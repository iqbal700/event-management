# 📅 EventMaster

**EventMaster** is a full-stack event management platform designed to help users discover, track, and manage local and international experiences. Built with the **MERN stack** and **Next.js**, it offers a seamless, responsive interface for both event organizers and attendees.

---

## ✨ Key Features

*   **User Authentication**: Secure login and registration powered by Firebase Auth.
*   **Dynamic Event Discovery**: Browse a variety of events like Music Festivals, Art Workshops, and Culinary Expos.
*   **Responsive UI**: A modern, mobile-friendly design built with Tailwind CSS and DaisyUI.
*   **Protected Management**: Logged-in users can access specific features to manage their registrations or events.
*   **Interactive Design**: High-performance navigation with Next.js and engaging hover effects on event cards.
*   **Real-time Feedback**: Custom error handling for authentication failures to improve user experience.
